Utility Shell
-------------

A Docker image containing some useful debug/troubleshooting utilities for a shell. Available via docker.io/makanab/utilityshell
- openjdk 8
- curl
- dnsutils 
- ping
- telnet
- tcpdump
- screen
- vim
- openssl
- netcat
- socat
- gron
- jmxdump
- kafkacat (with zstd)
- kafka 2.12-2.8.0
- apache2-utils (apachebench etc)
- jq
